# 🔧 Step 4. 플랫폼 특장점 & 부품 Teardown (Hardware Teardown)

AI 서버는 엄청난 전력(단일 랙 40kW~120kW)을 소비하며 이에 따르는 열을 해소하기 위한 초고성능 통신 네트워킹과 특수 인프라의 결합체입니다. 하이엔드 AI 서버 랙 내부의 핵심 물리적 컴포넌트를 분해하여 분석합니다.

---

## 📐 AI 서버의 구조적 분류 (OAM & UBB)
범용 데이터센터 하드웨어 표준인 OCP(Open Compute Project) 가이드에 기반하여 AI 가속기와 마더보드는 모듈 형태로 표준화되어 조립됩니다.

```
+-------------------------------------------------------------+
|                AI 서버 시스템 (System Chassis)              |
|                                                             |
|   +-----------------------------------------------------+   |
|   |         UBB (Universal Baseboard / 기판)             |   |
|   |                                                     |   |
|   |  [OAM 1]  [OAM 2]  [OAM 3]  [OAM 4] ... [OAM 8]     |   |
|   |  (GPU 모듈) (GPU 모듈) (GPU 모듈) (GPU 모듈)  (GPU 모듈)   |   |
|   +-----------------------------------------------------+   |
|                              ^                              |
|                              | PCIe Gen5 / NVLink           |
|                              v                              |
|   +-----------------------------------------------------+   |
|   |                  CPU 호스트 보드                    |   |
|   |     (Intel Xeon / AMD EPYC 및 DDR5 시스템 메모리)    |   |
|   +-----------------------------------------------------+   |
+-------------------------------------------------------------+
```

1. **OAM (Open Accelerator Module):**
   * 단일 GPU 칩셋 및 HBM 메모리가 회로와 함께 결합된 메인 보드형 모듈 부품. (예: NVIDIA H100 SXM5 모듈, AMD MI300X OAM).
2. **UBB (Universal Baseboard):**
   * 8개의 OAM(GPU 모듈)이 장착되는 거대한 베이스보드 기판. OAM 간 고속 통신 회로가 촘촘히 내장되어 있어 신호 손실을 최소화하는 고다층 초고속 기판(Ultra-high layer PCB) 필수 채택.

## 💧 액체 냉각 (Liquid Cooling) 핵심 구성 부품 Teardown
서버당 10kW 이상의 전력을 소모하는 Blackwell 세대부터는 공기 팬으로 열을 식히는 공랭(Air Cooling)의 물리적 한계를 넘어, **액체 냉각(Liquid Cooling)** 시스템이 필수 인프라로 자리잡았습니다.

```
 [외부 냉각탑] <--> [CDU (냉각 제어 분배 장치)] <--> [매니폴드 배관] <--> [콜드 플레이트 (GPU 칩 접촉)]
```

* **콜드 플레이트 (Cold Plate):**
  * GPU 및 HBM 다이와 다이렉트로 접촉하는 구리 재질의 특수 워터블록. 미세 핀 구조(Micro-fin)를 통해 내부를 순환하는 냉각수로 열을 신속히 전달.
  * *대표 공급사:* Auras, Cooler Master, Boyd.
* **매니폴드 (Manifold):**
  * 랙 캐비닛 수직 방향으로 설치되어 각 서버 슬롯 트레이로 냉각수를 고르게 분배 및 수거하는 이중 배관 샤프트.
  * *대표 공급사:* Volex, Nidec.
* **CDU (Cooling Distribution Unit / 냉각분배장치):**
  * 랙 내부의 냉각수 압력, 유량, 온도 및 수질을 미세 제어하고 외부 빌딩 냉각망과 열 교환을 수행하는 액체 냉각 계통의 심장부.
  * *대표 공급사:* Vertiv, nVent.
* **QD (Quick Disconnect / 퀵 커넥터):**
  * 서버 유지보수 시 냉각수 누수(Leak) 전혀 없이 원터치로 냉각 배관을 안전하게 탈착할 수 있도록 밀폐 밸브가 내장된 특수 금속 커넥터. 극도의 내구성 요구.
  * *대표 공급사:* Parker Hannifin, Staubli.
